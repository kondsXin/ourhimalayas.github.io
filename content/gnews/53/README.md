面对本次在加拿大十个城市同天进行的示威集会，盛况空前之下，多伦多枫叶农场依惯当仁不让，再度热血地战斗在此次“反疫苗护照”游行活动的最前沿，以力图让更多人了解新中国联邦拯救人类的天赋使命和爆料革命彻底消灭中共的根本立场。同时，借此全球行动之良机，有效宣传我们用以揭露“中共病毒”真相及其疫苗危害和传播解药信息的三大媒体平台：盖特、GTV和Gnews。
