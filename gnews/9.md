```
【编者按】中共海军正着眼于使用海外港口，包括大西洋港口。华盛顿智库战略与国际研究中心 (csis) 已经确定了中共参与的撒哈拉以南非洲近 50 个港口，有的正在建设中。 csis 表示，中共可能会利用它们“扩大其军事和政治影响力”，引起美国的国家安全担忧。《经济学家》周刊12月11日发表了这篇文章。
```

![](https://assets.gnews.org/wp-content/uploads/2021/12/1-161.png)

```
在过去十年中，中共对从石油到铜的进口需求不断增长，以及海外商业活动的扩张，加剧了其对保持贸易路线安全的担忧。在病毒大流行爆发之前，它在国外有 150 万工人。中共军事白皮书强调，海军需要保护这种“海外利益”。
```



```
但可能还有其他考虑。美国海军战争学院的瑞安·马丁森写道，中共军事分析人士公开主张进入远离家乡的水域，以转移美国对中共水域的注意力和资源。他注意到北京大学胡波的一本书，其中作者承认中共海军无法与美国争夺遥远海域的霸权，但暗示它可以“牵制”一些可能被派往东亚的美国军队。马丁森先生还引用了中共第一艘航空母舰辽宁号的舰长刘哲，他在 2017 年的一篇广为人知的文章中写道：“我们离领海越远，我们身后的祖国就越安全。”建立面向大西洋的基地肯定会引起美国的注意。
```



```
但马丁森在海军战争学院的同事艾萨克·卡登指出，中共领导人并没有重复有关扣留美国船只的强硬言论。他们宁愿在不与美国对抗的情况下建立基地。碰巧的是，大多数国家都不愿意通过接纳中共军队来激怒美国。同意中共基地将是一种交易安排，但对提高东道国的安全无济于事。与美国不同，中共不赞成结盟，并且厌恶在其他国家的冲突中发挥任何战斗作用。
```



```
几内亚湾将是寻找基地的合乎逻辑的地方。海盗在那里是一个祸害，就像在吉布提附近的亚丁湾一样，因此中共可以合理地争辩说需要它的帮助。一家中共公司在赤道几内亚的巴塔建造了一个深水商业港口。它可以适应中共军舰。而赤道几内亚的家族式独裁政权恰好是中共在非洲做生意时惯用的那种：野蛮、腐败和金融混乱。国际货币基金组织于 2019 年批准了一项救助计划。
```



```
副总统特奥多罗·恩圭马·奥比昂·曼格 (Teodoro Nguema Obiang Mangue) 是总统的儿子，他在 10 月接待了美国特使费纳先生，这出人意料地逆转了命运。近年来，美国及其盟友从他那里没收了据称是不义之财的奢侈品（美国表示，它正在利用所得收益为该国的 covid-19 疫苗和其他医疗援助提供资金）。副总统在社交媒体上发布了会议的视频片段，并在另一段视频中对他在访问中收到的外交礼物表示感谢：一个盖有总统印章的银盘。
```



```
卡登先生说，中共在巴塔的基地可能不会对美国构成严重威胁，除非它包括一个供中共潜艇使用的设施。但是建造一个几乎不可能隐藏。赤道几内亚不太可能让中共这样做而冒着激怒美国的风险。没有明确的证据表明中共有任何此类意图。
```


在费纳先生访问一周后，中共国家主席习近平与他的同行特奥多罗·奥比昂·恩圭马·姆巴索戈（Teodoro Obiang Nguema Mbasogo）总统通了电话。中共官员表示，奥比昂先生将中共描述为赤道几内亚“最重要的战略伙伴”。 12 月 7 日，他的副总统儿子在推特上回应了《华尔街日报》关于该基地的报道。 “中共是友好国家和战略伙伴的典范，但目前还没有这样的协议，”他写道。他很快补充道：“还要记住，赤道几内亚是一个主权独立的国家，可以与任何友好国家签署合作协议。”美国官员可能想安排一次后续访问。
