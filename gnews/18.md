华盛顿（TND）——据报道维里塔斯项目（ Project Veritas ）核实了一封以前未公开的拒绝信，该拒绝信来自一家政府机构，该机构拒绝向生态健康联盟（EcoHealth Alliance）提供资金，原因是未能评估他们寻求的在武汉病毒研究所进行的研究中，功能增强性研究的潜在风险。

生态健康联盟获得了国家过敏症和传染病研究所 (NIAID) 数百万美元的纳税人资助的赠款，用于在武汉病毒学研究所进行研究，研究冠状病毒如何从蝙蝠传播到人类。

批评者认为，受资助的研究包括有争议的功能获得性实验，但美国国立卫生研究院（ NIH ）主任弗朗西斯·柯林斯（ Francis Collins ）和 NIAID 主任安东尼·福奇（ Anthony Fauci ）博士等政府官员坚持认为，功能增强性研究并没有由政府花费一分钱。

来自肯塔基州的共和党参议员兰德·保罗（ Rand Paul ）在 5 月的参议院听证会上问福奇：“你能想象如果 SARS 病毒被提取并添加了病毒蛋白 – 刺突蛋白 – 那是否会被意外释放。”

福奇在听证会上回答道，“NIH 从来没有，现在也没有资助武汉研究所的功能增强性研究”。

维里塔斯项目报告称已获得，并核实了美国国防高级研究项目局 ( DARPA ) 拒绝向生态健康联盟提供资金的信函。根据维里塔斯项目的说法，拒绝信称该机构不会向生态健康联盟提供资金，因为其“提案未提及或评估功能获得性(GoF) 研究的潜在风险”。

维里塔斯项目还报告说已经获得了美国海军陆战队少校约瑟夫·墨菲（ Joseph Murphy ）写给国防部监察长的第二份报告，他也是 DARPA 的前研究员。

维里塔斯项目写道，“墨菲少校的报告继续详细说明了对中共病毒功能获得性计划、文件的隐瞒、抑制潜在的治疗药物（如伊维菌素和羟氯喹）以及 mRNA 疫苗的极大关注”。

维里塔斯项目表示已联系 DARPA 并与该机构的通讯主管贾里德·亚当斯（ Jared Adams ) 进行了交谈，当被问及为什么这些文件没有公开并可能消失时，他回答说：“这对我来说听起来很不正常”。“如果有什么是机密的，那么它应该被适当地标记，”亚当斯补充道。“我对放在机密地方的无标记文件一点也不熟悉，不熟悉。”

维里塔斯项目的创始人詹姆斯·奥基夫( James O’Keefe  ) 在一段视频中问道，如果国防部认为这项研究太危险而不可进行，为什么“ NIH、NIAID 和生态健康联盟会顾一切地忽视所涉及的风险？”
![](https://assets.gnews.org/wp-content/uploads/2022/01/image004-1.jpg)
国家事务部节目 ( TND ) 联系了 DARPA 和生态健康联盟，以确认维里塔斯项目所说的信件的真实性。生态健康联盟没有立即回应，但亚当斯告诉 TND，由于联邦法规，DARPA 无法确认文件的真实性。
