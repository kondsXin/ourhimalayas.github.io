自俄罗斯入侵以来已经过去了两个月，事实上越来越多的人从避难地波兰返回乌克兰，这个数字是大约每天2万人。尽管俄罗斯继续在各地发动袭击，到底为什么呢？
 
在波兰首都华沙的火车站，与返回乌克兰的人交谈，提出了以下4个问题，并将介绍三位受访者的回答。
 
1. 在哪里避难生活 ？
2. 为什么决定避难撤离 ？
3. 为什么要回乌克兰 ？
4. 回国后有担心吗？

**我不能离开我的父亲**
 
**伊丽莎白（22岁）从东部顿涅茨克州撤离**

1. 在哪里避难生活 ？

大约一个月前，我与母亲和9岁的妹妹从乌克兰东部的顿涅茨克州撤离。经由波兰前往塞浦路斯的亲戚家避难，但这次我决定独自返回乌克兰。
 
2. 为什么现在回国？
 
几天来我和留在乌克兰的父亲失去了联系，我不知道他在哪里，在做什么？我不能把父亲一个人留在乌克兰。如果父亲参军，我无能为力，但无论如何想见到父亲，带他去乌克兰安全的地方。
 
3. 当决定撤离时，您家周围的情况如何？
 
我居住的地区不受俄罗斯军队控制。即便如此，我家周围也遭到了轰炸，轰炸的声音很可怕，所以决定先把家人疏散到安全的地方。妈妈不愿意我一个人回到乌克兰，但我做出解释并说服了妈妈。塞浦路斯是一个美丽、和平和安宁的城市，但在乌克兰士兵们仍然处于悲惨境地，担心所有人，无论我身在何处，脑海里都是乌克兰。
 
4. 回国后有担心吗？
 
我什么时候可以找到父亲，又该如何解决？太多未知的情况，我不知道什么时候能回来，所以只有一张单程票。但现在我不在乎未来的生活会是什么样子，我只担心我的家人。此外，我没有男朋友，所以比别人少一个问题。不管发生什么事，我都会接受这一切，因为这是我的生活。如果我能帮助父亲，别无选择会尽我所能，我不在乎。
 
**我找不到工作**
 
**伊琳娜（理发师）从首都基辅撤离**

1. 在哪里避难生活 ？

大约一个月前，我从首都基辅撤离到波兰，和一个波兰熟人住在一起。在波兰非常感谢大家的善意，我正在找工作，但由于语言障碍找工作很难。
 
2. 为什么现在回国？
 
怀念我的家乡，最近基辅很平静，所以我要回家。另外，在这里找工作真的很难。在任何国家生活都需要钱，很多人来到波兰。在工厂我也有一份制造的工作，但工作很艰苦。即使在战争期间，基辅也有很多工作。任何想工作的人都可以赚到钱，比波兰更容易，因为了解我们的国家。当然，也不是那么好。
 
3. 当决定撤离时，您家周围的情况如何？
 
我住在基辅市中心，警笛声不断响起，导弹从头顶飞过。太可怕了。人们每天都在避难所里，我不能呆在家里。尤其是战争开始后的第一周最害怕，因为不知道会发生什么？市中心的两栋公寓楼遭到破坏。导弹在飞，所以我想我必须逃到某个地方。
 
4. 回国后有担心吗？
 
如果不得不再次撤离，我将别无选择，只能返回波兰的某个地方，比如华沙，我希望这场战争能早日结束。无论如何 都和以前不一样了，随着经济危机发生将来会更加困难。乌克兰有这么多房屋被毁，我真诚地希望其他国家能够协助我们重建家园。
 
**公司叫我回来**
 
**Aro** **na Habriluk(** **会计师）从西北部日托米尔州撤离**

1. 在哪里避难生活？

3月5日左右，我和兄弟的妻儿来到波兰，在熟人帮助下住进一个普通家庭里。一切都很好，我非常感谢波兰人民。
 
2. 为什么现在回国？
 
我有点累了，所以想回家。住在另一个家庭不能像在家里一样生活。寄宿的家庭里有三个孩子，我们累了。也许战争会持续数月甚至一年，我们没有足够的钱或时间呆在这里。另外，我接到公司来的电话，说：“拜托回来工作吧。” 可以在网上办公，我没有辞职，但如果在这里待得太久，我想可能不会在日托米尔找到工作。
 
3. 当决定撤离时，您家周围的情况如何？
 
炸弹落在街道中央，其中一枚落在学校被完全摧毁，离我工作的地方大约500米。整日都能听到飞机低空飞行的声音，所以我们决定撤离到波兰。
 
4. 回国后有担心吗？
 
寄宿的家庭说：“你可以回来。”但我只是希望同样的事情不要再发生了。在日托米尔发生爆炸的事件并不多，也许有东西从天而降，但我们的街上没有俄罗斯士兵。也许能够继续在线工作几周，但这总是很困难。由于在国有企业工作，需要很多文件资料，我被要求每天早上8点上班。如果不返回 将无法在此期间领取工资。那样的话，我必须辞掉工作，在波兰找份新工作，然而学习波兰语需要很长时间。此外，我不想留在这里，因为我的父母和家人仍然在乌克兰。
 
**在火车站，挤满了回乌克兰的人**
 
这一次，只有几个人接受采访，华沙的火车站挤满了乘坐卧铺火车和公共汽车去乌克兰的人。起初，我还在想为什么要回到危险的乌克兰，但正如被告知的那样，我意识到随着避难生活时间的延长，和不得不回家的烦恼存在着冲突。
