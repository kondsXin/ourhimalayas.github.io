牛津经济研究院预测，如果房地产巨头恒大的债务问题蔓延到其他经济领域，中国的经济增长很可能将在 2022 年降至 1%。
```



```
这将是自 1976 年以来最弱的国内生产总值扩张，那一年中国的文化大革命以共产党独裁者毛泽东的逝世告终。

```
按照这个速度，经济的增长速度也将不及 2020 年中国进入严格封锁时 2.3% 的速度的一半。

```

```



```
一位国家安全专家表示，如果经济增长下滑，习近平作为中国国家主席的地位可能会受到共产党内部的威胁。作为“共同富裕”计划的一部分，这位威权领导人对亿万富翁的镇压可能会破坏经济增长，而对基本出口的限制可能会使澳大利亚等其他国家建立自己的供应链，以实现多元化，远离中国
```



```
他对澳大利亚《每日邮报》说：“自毛泽东去世以来，共产党的合法性来自它为中国人带来的经济增长。”
```



```
“党为继续掌权而达成的协议是，‘你让我们继续掌权，让我们远离政治和其他敏感话题的讨论，作为回报，中国经济将为你带来财富。’”
```



```
舒布里奇先生表示，自 2013 年以来掌权的习主席更有可能面临来自中国共产党内部的挑战，而不是由于经济大幅放缓而引发民众造反。

```

```
虽然许多共产党官员会感到恐惧，但习主席最终也可能误判自己的政治局势，因为他的领导人两届任期限制已于 2018 年取消。
