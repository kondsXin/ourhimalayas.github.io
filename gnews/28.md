**中共国政府周三批准了稀土矿产公司的合并，以形成官方媒体《环球时报》所描述的“巨无霸”公司，强大到足以控制重要矿产的全球供应链。**

新的大型企业将由中共政府下属的国有资产监督管理委员会（SASAC）直接控制。该公司将涵盖中国约70%的稀土行业。


```
一位不愿透露姓名的业内人士表示，新公司有望“优化资源，确保供应安全，并在稀土产量和出口量方面执行更严格的规定。”
```



```
《环球时报》称，这一消息是对美国总统乔拜登如一记耳光，拜登曾对中共国在稀土供应链中的主导地位表示担忧——尤其是考虑到这些金属对风能、太阳能和电动汽车的重要性，拜登政府希望花费大量美国纳税人的钱购买或补贴这些产品。
```



```
《环球时报》嘲笑拜登认为他可以通过与澳大利亚供应商达成交易或开发美国稀土矿床削弱中共国的主导地位，显然时间太迟了。
```


相反，中共国分析人士表示，北京创建的新企业巨头将赋予中共国“在谈判桌上的影响力和议价能力”产生更深远的意义，并保护中共国免受“西方的欺凌企图”。

《环球时报》指出，在武汉冠状病毒大流行期间，由于产量减少，稀土价格飙升，国资委批准的大规模合并可能会进一步推高价格。

中共国工业对稀土的需求如此之高，在占了世界供应的绝大份额后，还要进口更多的矿物。缅甸的政治动荡推动了价格上涨，因为缅甸是向中国冶炼厂出口金属的主要源头。人权倡导者抱怨说，中共国对金属的大量采购正在让目前统治缅甸的军政府更强大。

中国行业消息人士周四对英国《金融时报》表示，北京希望加强对稀土的控制以稳定价格，将“价格剧烈波动导致终端用户头疼”的风险降至最低。

该理论认为，如果相互竞争交付金属的公司减少，价格将停止大幅波动。反过来，这将使严重依赖稀土的行业更容易建立稳固的长期商业模式和安全投资。

“鉴于稀土的战略重要性，我们不能让市场力量决定稀土的价格。考虑到稀土的战略重要性，我们不能让市场力量决定稀土的价格。” 该行业消息人士表示。

其他分析师告诉《金融时报》，自川普任期与美国的贸易战初期，中共国就一直在计划破坏美国开发替代供应商的努力。

中共国作为全球最大稀土供应商国，控制了稀土的供应链。中共希望加强对于稀土的管控，从而影响西方世界国家的工业链，这再次证明了中共希望控制世界话语权和统治世界的野心。
